# Secure API Key Guide

To allow other people (like your teachers or friends) to use the Quiz Generator on scanned PDFs without needing their own Gemini API key, you can embed your key securely.

Because this GitHub repository is public, **you cannot paste your raw API key directly into the code**. If you do, bots will steal it.

Instead, we use **AES-256-GCM encryption**. You will encrypt your API key locally using a **Passcode**, and put the *encrypted gibberish* into the code. When someone visits the site, they type the Passcode, and the site decrypts the key in their browser memory.

### Step 1: Install Python Cryptography
Open your terminal and install the required library:
```bash
pip install cryptography
```

### Step 2: Run the Encryption Script
Run the helper script included in the root of the project:
```bash
python encrypt_api_key.py
```

It will ask you for two things:
1. **Your raw Gemini API Key**: (e.g., `AIzaSy...`)
2. **A Passcode**: Choose a memorable password (e.g., `hebrew-quiz-2026`).

*Note: Choose a strong passcode! Anyone who guesses the passcode and looks at the source code can decrypt your API key.*

### Step 3: Update `generator.js`
The script will output a block of code that looks like this:
```javascript
    const EMBEDDED_KEY = {
        encryptedKeyB64: '...gibberish...',
        ivB64: '...gibberish...',
        saltB64: '...gibberish...'
    };
```

Open `generator.js`. At the very top (around line 6), replace the empty `EMBEDDED_KEY` block with the new one generated by the script.

### Step 4: Commit and Push
Save `generator.js` and push it to GitHub:
```bash
git add generator.js
git commit -m "chore: embed encrypted api key"
git push origin feat-site
```

### Step 5: How to use it
Once deployed, anyone who visits the live site can simply type your chosen Passcode into the "Passcode" box and click "Run Parse". They do not need to fill in the "Gemini API Key" box at all.

If they ever want to use their own key, they can leave the Passcode blank and paste their own API key into the top box.
